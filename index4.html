Etapes de paramétrage RT CISCO 2811

Mise en oeuvre initiale

1 Passage en mode privilégié puis configuration :
> enaenable
# conf tconfigure terminal
(config)#
2 Changement du nom :
(config)# hostname nom-du-routeur
3 Mettre une bannière :
(config)# banner & message de la banière ...
(config)# ... sur plusieures lignes ...
(config)# ... &
4 désactiver le service Cisco descovery protocol :
(config)# no cdp run
5 désactiver la résolution de nom :
(config)# no ip domain lookup
6 Désactiver le serveur HTTP :
(config)# no ip http server
7 Synchoniser la console :
(config)# line console 0
logging synchronous
exit

Configuration d'une interface non tagger

1 Configuration de l'interface:
(config)# int nom-de-l'interface (ex: f0/0)
(config-if)# ip address @IP mask
(config-if)# desc description-de-l'interface
(config-if)# no shut
(config-if)# no ip proxy-arp
(config-if)# exit
#

Configuration d'une interface tagger

1 Configuration du protocole 802.1Q :
(config)# int nom-de-l'interface-(ex: f0/0)
(config-if)# no ip proxy-arp
(config-if)# no shut
(config-if)# int fx/x.+n°-du-vlan
(config-subif)# encapsulation dot1q n°-du-vlan
2 Configuration de l' @IP:
(config-subif)# ip address @IP mask
(config-subif)# no ip proxy-arp
(config-subif)# exit
#

Ajouter une route statique

1 Ajouter une route :
(config-if)# ip route @RSO mask @nextGW distance

Sauvegarder la configuration

1 Copie la configuration de la ram dans le fichier start-config :
# copy run-config start-config

Charger une configuration

1 Chargement en mode TFTP :
# copy tftp start-config @IP-serveur Nom-du-fichier.cfg

Transferer la configuration

1 Tranfert en mode TFTP :
# copy start-config tftp
Address or name of remote host []? @IP-serveur
Destination filename [xxxxx]? Nom-du-fichier.cfg

Faire un redémarrage en configuration usine

1 Faire un redémarrage en configuration usine
# erase start-config
# reload
