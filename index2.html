Etapes de paramétrage SW ALLIED AT8624

Mise en oeuvre initiale

1 Parametrage du nom machine :
SET SYStem NAMe=nom du switch
2 Chargement de la doc :
SHow FILe=*.hlpPour afficher la liste des fichiers de type .hlp
SET HELP=nom du fichier.hlpPour charger le fichier

Parametrage des VLAN's

1 Creation du VLAN :
CREate VLAN=nom du vlan VID=n° du VLAN (2...4094)
2 Ajouter un port sur un VLAN :
ADD VLAN=n° du VLAN (2...4094) POrt=n° des ports (ex: 2-3,5)
3 Ajouter un port tagger sur un VLAN :
SET VLAN=n° du VLAN (2...4094) POrt=n° du port FRAme=TAGged

Créer une @IP d'administration

1 Ajouter une @IP :
ADD IP INTerface=vlan+n° du VLAN IPaddress=@IP MASK=mask en deci.
ENAble IP Pour l'activer

Attribuer une GW

1 Création d'une route par défaut :
ADD IP ROUte=0.0.0.0 MASK=0.0.0.0 INTerface=vlan+n° du VLAN NEXThop=@IP de la GW

Paramétrage d'un mirroring

1 Créer un mirroring :
SET SWItch MIRRor=n° du port de destination
SET SWItch POrt=n° du port écouté MIRRor=both

Sauvegarder la configuration

1 Créer levenfichier de configuration :
CREate CONfig=nom du fichier.cfg SET

Charger une configuration

1 Chargement en mode TFTP
METhod=TFtp FIle=nom du fichier.cfg SErver=@IP du serveur

Transferer une configuration

1 Tranfert en mode TFTP :
UPLoad METhod=TFTP FIle=nom du fichier.cfg SErver=@IP du serveur

Faire un redémarrage en configuration usine

1 Faire un redémarrage en configuration usine
SET CONfig=nonePour décharger le fichier du boot
SHow FIle=*.cfgPour afficher la liste des fichiers de type .cfg
DELete FIle=nom du fichier.cfgPour effacer le fichier de configuration
RESTART REBoot
